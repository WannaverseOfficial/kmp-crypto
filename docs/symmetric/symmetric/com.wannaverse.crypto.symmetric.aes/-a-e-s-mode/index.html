enum AESMode : Enum<AESMode>

Enum class representing the different modes of AES cipher used in various encryption schemes.

Each mode corresponds to a specific configuration of AES encryption, including the padding scheme and encryption type. The modes in this enum can be used to specify the cipher mode when performing encryption or decryption with AES.
